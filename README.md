# Feedback Hub - Sentiment Analysis Dashboard

## Project Overview
Feedback Hub is a full-stack application designed to collect customer feedback, analyze sentiment using rule-based logic, and categorize reviews into themes (Comfort, Durability, Appearance).

## Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML, JavaScript (Fetch API), Chart.js
* **Data Storage:** In-memory list (Python)

## Key Features
1. **Feedback Submission API:** Accepts and stores feedback.
2. **Sentiment Analysis:** Detects Positive/Negative sentiment using keyword logic.
3. **Theme Detection:** Identifies themes like Comfort and Durability.
4. **Insight Generator:** Suggests actions based on data trends.

## How to Run
1. Start the backend: `python app.py`
2. Open the frontend: `open index.html`
